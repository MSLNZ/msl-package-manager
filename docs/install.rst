Install the MSL Package Manager
===============================

To install the **MSL Package Manager** run::

   pip install msl-package-manager

Dependencies
------------
* Python 2.7, 3.3+
* pip_
* colorama_

.. _pip: https://pypi.python.org/pypi/pip
.. _colorama: https://pypi.python.org/pypi/colorama
