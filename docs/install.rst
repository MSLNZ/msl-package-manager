Install the MSL Package Manager
===============================

To install the **MSL Package Manager** run:

.. code-block:: console

   pip install msl-package-manager

Dependencies
------------
* Python 3.8+
* setuptools_
* colorama_

.. _setuptools: https://pypi.org/project/setuptools/
.. _colorama: https://pypi.org/project/colorama/
